# CogniSimple - نمونه اولیه v0.1

اپلیکیشن ارزیابی شناختی توجه با هوش مصنوعی  
**محور فراخوان:** نوروانفورماتیک و فناوری‌های ارزیابی شناختی

## تیم
- **مجتبی صابری** (دانشجوی دکتری علوم اعصاب شناختی رایانشی) - مدل AI
- **مریم براتی** (کارشناسی ارشد فناوری اطلاعات) - توسعه Flutter

## قابلیت‌ها
- آزمون Stroop (فارسی)
- تحلیل توجه با Random Forest
- گزارش شخصی‌سازی‌شده (دقت، زمان، سطح توجه)
- بدون نیاز به سرور

## اجرا
```bash
# مدل هوش مصنوعی
cd python_model
python train_model.py

# اپلیکیشن موبایل
cd flutter_app
flutter pub get
flutter run
